# n8n Local Installation Guide (Node.js Only)

This guide explains how to install and run **n8n locally on your desktop using Node.js**.

All commands are written for **Bash / Terminal / Command Prompt**.

---

# Step 1: Install Node.js

Download and install Node.js (version 18 or higher) from the official website:

https://nodejs.org/

---

# Step 2: Verify Node.js & npm Installation

Open your terminal and run:

bash:
node -v
npm -v

---

# Step 3: Install n8n Globally

Use npm to install n8n:

bash:
npm install n8n -g

Wait for the installation to complete.

---

# Step 4: Start n8n

Run the following command:

bash:
n8n

n8n will start a local server.

---

# Step 5: Access n8n in Browser

Open your web browser and visit:

http://localhost:5678

---

# Step 6: Stop n8n Server

To stop the server, press:

CTRL + C

---
